# DataWiz Club Website

A full-stack web application developed to manage and promote activities of the DataWiz Club. The platform centralizes event updates, announcements, and member engagement, improving visibility and organization.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Event management and announcements
- Member interaction and updates
- Admin functionality for content control
- Responsive and modern UI

## Purpose
Built to solve the lack of a centralized system for club operations and event promotion, enabling better coordination and outreach.

## Setup
1. Clone the repository  
2. Install dependencies for both client and server  
3. Configure environment variables (MongoDB URI)  
4. Run frontend and backend servers  

```bash
npm install
npm start
```
